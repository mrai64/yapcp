Cursor AI √® un ambiente di co-sviluppo, dove l'ai
da una mano, ma vs code ha le estensioni e particolarmente
Markdown calendar che uso molto. Ritorno a VS Code

Cosa chiedere all'archivio per creare la lista
di tutti i lavori partecipanti,
in ordine di sezione, in ordine di nazione
in ordine di cognome e nome
in ordine di presentazione
se hanno avuto ammissione
se hanno avuto premi assegnati

```sql
SELECT  pcp_contest_works.section_id,
        pcp_countries.flag_code,
        pcp_user_contacts.country_id,
        pcp_user_contacts.last_name,
        pcp_user_contacts.first_name,
        pcp_contest_works.portfolio_sequence,
        pcp_works.work_file,
        pcp_works.title_en,
        pcp_contest_works.is_admit,
        COALESCE(`pcp_contest_awards`.award_name, '') AS award_assigned
   FROM pcp_contest_works
LEFT JOIN pcp_user_contacts on (pcp_contest_works.user_id = pcp_user_contacts.user_id)
LEFT JOIN pcp_works on (pcp_contest_works.work_id = pcp_works.id)
LEFT JOIN pcp_countries on (pcp_user_contacts.country_id = pcp_countries.id)
LEFT JOIN pcp_contest_awards on (pcp_contest_awards.winner_work_id = pcp_contest_works.work_id)
WHERE pcp_contest_works.contest_id = 'e8ac5674-c3d1-4afa-adaf-a7d5ed82d292'
ORDER BY  pcp_contest_works.section_id,
          pcp_user_contacts.country_id,
          pcp_user_contacts.last_name,
          pcp_user_contacts.first_name,
          pcp_contest_works.portfolio_sequence;
```

## Rifare la lista delle librerie SBOM

* Serve andare in github.
* Selezionare il repository.
* Nella barra di navigazione scegliere Insights.
* Scegliere dalla colonna a sinistra / start la voce Dependency graph.
* Scegliere il pulsate in alto a destra / end il pulsante EXPORT SBOM.
* Viene scaricato un file che va riformattato e rinominato sbom.json.
* il file sbom.json va postato all'interno del repository nella cartella radice
e lasciato lì, fino al prossimo composer require / remove / update.

## **Problema delle belongsTo**

Per adattarmi a §PSR-12 ho scritto molte funzioni di relations
ma pare che abbia invertito il concetto di foreign key e di owner key

```php
Class UserContact
...
// sbagliato 
    public function country()
    {
        $country = $this->belongsTo(
            related: Country::class, //   countries.
            foreignKey: 'id', //          NON E' ❌ countries.id
            ownerKey: 'country_id' //     NON E' ❌ user_contacts.country_id
        );
        return $country;
    }

// giusto
    public function country()
    {
        $country = $this->belongsTo(
            related: Country::class, //   countries.
            foreignKey: 'country_id', //  user_contacts.country_id
            ownerKey: 'id' //             countries.id
        );
        return $country;
    }

```

## main

Rimossa la sezione /docs di fatto sempre abbozzata e che viene
sostituita dalla sezione /resources/docs/dev/man

Creata una introduzione ai database e un command per generare
un elenco delle tabelle, dei campi che le compongono, delle relazioni fk
con altre tabelle.

Il template della pagina diario viene sempre creato su main,
anche se poi il lavoro va in un branch diverso. Questo consente
di mantenere allineata la pagina markdown in modo da semplificare
il merge automatico.

* creazione data su main;  
  anche la data di domani, con template vuoto
* copia pagina markdown
* commit push
* cambio branch
* creazione data su branch
* incolla pagina markdown
* prosecuzione lavori
